Flowaid
Flowaid is a digital platform designed to improve access to menstrual health support for girls in low-income communities. The system focuses on accessibility by using simple technologies such as USSD, SMS, and a lightweight mobile web interface.
The aim is to reduce period poverty and improve school attendance by making information and support easier to access.
Problem
Many girls in underserved communities face:
Limited access to menstrual products
Lack of accurate health information
Stigma around menstruation
Missed school days
Most digital solutions require smartphones and internet access, which excludes many users.
Solution
Flowaid provides:
USSD and SMS access for basic phones
A mobile-friendly web interface
Menstrual health information
A simple support request system
Transparent tracking of aid distribution
The platform connects beneficiaries, local partners (schools/NGOs), and donors in a structured way.
Core Features
Basic phone support (USSD/SMS flow)
Lightweight web dashboard
Support request submission
Aid tracking and reporting
Privacy-focused user handling
Tech Approach (Planned)

Ethics & Privacy

Transparent reporting of funds
